Bachelor-Arbeit: Eiham Salieba

WebSpeech-API Tutorial

Speech input and output technologies in Web browsers offer an alternative interaction modality, which is particularly beneficial for visually impaired persons. But also multimodal user interfaces can be built with speech input and output as a component. Speech input and output technologies can also form the basis of conversational user interfaces. The goals of this thesis are twofold:

  • Explore the state of the art of speech input and output technologies in current Web browsers, focusing on Google Chrome and Mozilla Firefox
  • Develop two example scenarios of speech interactions in browsers, implemented as browser extensions
